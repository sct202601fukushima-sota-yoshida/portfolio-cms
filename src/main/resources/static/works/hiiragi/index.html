ひいらぎ不動産は杉並・新宿・江戸川・港区に強い地域密着型の不動産会社です。

賃貸住宅の管理から売買物件の仲介、リフォーム、保険の代理など
住まいに関するあらゆるご相談を承っています。

私たちの会社について

ひいらぎ不動産は、東京で45年以上愛され続けている会社です。

ひいらぎ不動産は1973年の創業から皆様のおかげで45周年を迎えることができました。

江戸川区での創業から初心を忘れず、下町の心意気を感じる人情味溢れる親身なお客様対応を徹底し、
これからも社員一同一丸となってお客様の満足のため努力を続けていきます。

コーポレートステートメントについて

コーポレートステートメント「単身でも、安心。単真でも、安心。」は弊社代表の父である江ヌ系 六郎が遺した言葉です。

創業時、江戸川区の下町のみで不動産業をしていた当初は田舎から進学のために上京する学生さんなど、単身者で何も持たずに上京する方のサポートに特に力を入れて営業していました。

「身も心も」落ち着いた状態で上京してもらうため「真心」を込めてサポートする。

この想いは、今でも弊社の社員一同に引き継がれ会社全体の指針として掲げられています。

お客様の声

松下隆司様

月々の家賃や立地面など、自分でもかなりわがままだと感じるレベルの条件にもかかわらず親身になって対応していただき、少し時間はかかりましたがひいらぎ不動産に対応していただいたおかげで希望通りの部屋に住むことができました。

担当していただいた田中さんのカバンには滋養強壮剤の空き瓶がたくさん入っていたのを見て非常に熱心に対応してくださっている証だと思い感動しました。

これから購入をした物件で生活するイメージをするとワクワクします。有難う御座いました。

佐竹絵里子様

内見時にいきなり聴診器を持ち出して四方八方から生活音が聞こえないか調べ始めたときは驚きましたが、実際に部屋を借りて住む私以上にこれからの生活が快適なものになるか真剣に考えて行動してくださる所を見て、全てひいらぎ不動産でお任せしたいと決心するきっかけになりました。

東京都内でここまで親しみやすく、親身に対応していただける不動産会社はひいらぎ不動産以外にないと思います。

非常にお世話になり感謝しています、ありがとうございました。

糖尿と痛風(梨本興業所属)様

金銭的な問題がなによりもネックで、どこの不動産会社でも内見にすら取り合っていただけない中、いわゆる「いわくつき物件」まで含めて快適に住める場所を探していただき、おかげさまでやっと2人で住める場所を見つけられました。

対応していただいた方がドイツ人の方だったので最初は少し不安もあったのですが、非常に気さくでとても親しみやすい対応をしてくださったのが印象深いです。

あのあとお祓いに行くと言っていましたが大丈夫だったのでしょうか？

スタッフ紹介

高橋伸治(営業部/部長)

壁と騒音問題のエキスパート。上京時に入居したアパートの壁が薄すぎて、しばらく隣人からの壁ドンに怯える日々を送る。

その経験から内見時はお客様より先に聴診器で壁の薄さをしつこく調べるなど、部屋選びには徹底したこだわりを持つ。

田中信行(営業部/WEBコンテンツ)

弊社WEBコンテンツ運営の責任者でありバラエティ系記事のリアクション担当。

体を張ったレビュー記事を担当した後は必ず社長にダミーの退職届をチラつかせて高級な焼肉を食わせろと要求する。

老化が酷く滋養強壮剤が友達。

シモン・アーレント(新宿店)

数年前にドイツから日本に移住し、ひいらぎ不動産で部屋を選んだことをきっかけに入社。

母国にいた頃より人との交流が増えたようで、キャラクターが年々パリピ気味になっている。最近のブームは寺社仏閣巡り。

佐々木ゆかり(営業部/法人担当)

事業用不動産に関するスペシャリスト。

法人及び個人投資家のお客様に対して不動産の価値を最大限に活かしたご提案を専門に行う。

弊社の高橋伸治から部長の座を奪おうと目論んでいる肉食系キャリアウーマン。

弊社スタッフについて

ひいらぎ不動産では多様性を受け入れ自由な社風を尊重するとともに、積極的な人材育成制度を設けることで宅地建物取引士やファイナンシャルプランナーはもちろん、1級建築士や福祉環境コーディネーターなどあらゆる分野に対応できる専門知識を持った人材育成に努めています。
